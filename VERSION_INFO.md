This is the initial stable release of Antika Deposu application with the following features:

- ✅ User registration and authentication

- ✅ Antique car parts request system
- ✅ Photo upload (camera & gallery)
- ✅ Admin panel with search and filters
- ✅ Request management

#### Platform Support
- ✅ iOS (React Native)
- ✅ Android (React Native)
- ✅ iOS (Flutter - alternative)
- ✅ Android (Flutter - alternative)

#### Backend
- ✅ Express.js REST API
- ✅ MongoDB database
- ✅ JWT authentication
- ✅ File upload support
- ✅ Email service integration
